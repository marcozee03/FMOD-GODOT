FmodEventEmitter2D
==================

**Inherits:** ``Node2D``

Emits Fmod Events and places them in the 2d world

.. rst-class:: classref-introduction-group

Description
-----------

FmodEventEmitter2D can be triggered with signals. Sounds will be played at the

emitters transform or it's parents.
